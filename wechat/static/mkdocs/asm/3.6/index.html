asmcmd管理模板

原创 2017-11-20 Oracle 宅必备

这个专题讲ASM相关的内容

这节讲ASM 的asmcmd命令

asmcmd命令是我们管理ASM的一个命令行工具

我们可以使用help dsget的形式查看帮助

通过grid用户进入

ASM模板相关命令

1.1 chtmpl

该命令改变一个模板的属性

ASMCMD [+] > chtmpl -G data --redundancy high --striping fine mytemplate

1.2 lstmpl

该命令列出所有磁盘组或者指定磁盘组的模板

1.3 mktmpl

该命令增加一个模板到磁盘组

ASMCMD [+] > mktmpl -G data --redundancy mirror --striping coarse mytemplate

1.4 rmtmpl

该命令从磁盘组删除一个模板

ASMCMD [+] > rmtmpl -G data mytemplate

具体请参考下面链接:

https://docs.oracle.com/cd/E11882_01/server.112/e18951/asm_util005.htm#OSTMG94659
